# senseCam
esp-idf project based on seeed studio xiao esp32-s3 sense cam

Quickstart:
  Download Esp-IDF
  Make sure it runs by activating it
  Clone this repo
  cd into it, with esp-idf activated in the terminal.
  run build process - if passes, upload directly to your board.

  Make sure to read your errors! They'll lead you in the right direction!
  If you need to change the config, run the menuconfig option and look for the setting you missed
